# Gift Tax Calculator (Python)

This is a small Python program I wrote to practice using conditional logic with real-world rules.

The program asks for the value of a gift and then calculates how much tax is owed based on different tax brackets.

# Why I Built This
I wanted a project that went beyond simple examples and forced me to think through how real rules (like tax brackets) translate into code.

# What This Program Does
- Takes a gift value as user input
- Checks which tax bracket the value falls into
- Calculates the correct tax amount
- Prints the result to the user

# Example Run
Value of gift: 12000
Amount of tax: 660.0

# What I Learned
- How to break a real problem into smaller logical checks
- How to use `if / elif / else` statements effectively
- How small math mistakes can affect results
- How to use Git and GitHub to publish a project

# Things I’d Improve Next
- Handle invalid input (negative numbers, non-numeric input)
- Format the tax output more cleanly
